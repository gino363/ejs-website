# Message App API

A simple full-stack message application built with Node.js, Express, and a basic JSON storage system.

## Features
- Send messages from frontend
- View messages via API
- Simple backend using Express
- Frontend interface with HTML/CSS/JS

## Tech Stack
- Node.js
- Express.js
- JavaScript
- HTML/CSS
- JSON file (temporary storage)

## How it works
- Frontend sends data (name + message)
- Backend receives it via POST request
- Messages are stored in a JSON file
- GET request returns all messages

## API Endpoints

GET /message  
Returns all messages

POST /message  
Adds a new message
